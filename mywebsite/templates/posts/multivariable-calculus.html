A couple of weeks ago, I revisited some old concepts from my multivariable calculus courses to refresh my memory. To demonstrate to myself that I fully understood topics like the multivariable chain rule, the Hessian matrix, and Lagrange multipliers, I created several Jupyter notebooks that show each concept in action.

Multivariable Calculus with Python

These notebooks, written using Python and Matplotlib, cover the following concepts:

  1. The Gradient
  2. Partial Derivatives and the Chain Rule
  3. Curvature, Divergence, and Curl
  4. The Laplacian
  5. The Jacobian
  6. Tangent Surfaces and The Hessian
  7. The Second Partial Derivative Test
  8. Lagrange Multipliers and Constrained Optimization

Rather than make a full post about them here, I figured it would be best to link to this collection of notebooks so that you can try them for yourself.
